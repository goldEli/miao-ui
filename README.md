### miao-ui 组件库

miao-ui 基于 React 开发的一套组件库

[查看文档](https://goldeli.github.io/miao-ui)

#### Install
```
  npm install @samuel.miao/miao-ui --save
```

#### Usage
```
// 加载样式
import '@samuel.miao/miao-ui/dist/index.css'

// 引入组件
import { Button } from '@samuel.miao/miao-ui'
```

#### Development

```
// dev
yarn storybook

// test
yarn test

// build 
yarn build

// publish npm
yarn publish
```
